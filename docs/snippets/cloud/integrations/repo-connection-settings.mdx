After the authentication, you need to fill in the following details:
- **Repository** - The full name of the code repo.
- _Optional_ **Environment base branch** - If you want Elementary to open PRs in a target branch different than default, detail the branch name here.
- _Optional_ **Project path** - If your dbt project isn't on the root directory of the repo, detail it's path here.
- _Optional_ **Update token** - When the github token expires, regenerate a fine-grained token and paste it here.
