Elementary provides monitoring of data quality metrics, freshness, volume and schema changes, including anomaly detection.
Elementary data monitors are configured and executed like native tests in dbt your project.

### Our tests

- Volume anomalies
- Freshness anomalies
- Column anomalies
- Dimension anomalies
- Schema changes
- JSON schema changes
- Exposure validation
- Python tests
