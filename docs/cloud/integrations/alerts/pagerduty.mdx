## What you can send alerts about

- Model run failures
- Failures and/or warnings of dbt tests (including Elementary dbt package)
- Failures and/or warnings Elementary Cloud test tests
    - Automated volume tests
    - Automated freshness tests
- dbt source freshness failures

## Enriching alerts

You can enrich your alerts by adding properties to tests and models in your **`.yml`** files. The supported attributes are:

- owner
- tags

Those enrichments are used for filtering alerts only.
